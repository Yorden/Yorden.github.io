Work

Desertman Drunkrun

A casual endless runner game I developed for Android using LibGDX. You must dodge obstacles and collect booze to survive. All levels and pickup placements are procedurally generated.

Palaver Server and Client

A chat app made for Android using Android Studio and Node.js. The app allows you to connect with random people in small 5 person chatrooms and talk anonymously. Features a sign in system, changeable user settings and chatroom creation and deletion.

Wordlike

A simple word game I made while working at Quill.org. The game is made to teach kids from 5th to 8th grade about synonyms through matching words. Players can see realtime player scores and are given challenges to accomplish every level. I used AngularJS and Firebase to creat this. The game is playable here.

Quartz

This is a 3D platformer I made in Unity. You play as a sphere who tries to roll around collecting gems while not falling into pits of lava. This game is only playable with an Xbox controller.

Topping of Death

A fun and simple game I made for the web browser using HTML 5 canvas. In game, you jump and try to shoot flies out of the sky, while trying not to get killed. The game is playable here.
